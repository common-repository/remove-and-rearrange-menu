== Description ==

Remove and Rearrenge Menu is free. This plugin allow you easily to delete your menu link or move your menu link current position to whenever want position. You have to delete any menu link via one click you dont want to open this menu for delete.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `remove_and_rearrange_menu.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

= Minimum Requirements =

* PHP 5.6 or greater is recommended
* MySQL 5.6 or greater is recommended
